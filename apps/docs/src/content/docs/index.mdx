## Quick install

<Tabs>
  <TabItem label="Playwright Reporter">
    ```bash
    npm install @lgtm/playwright-reporter
    ```
  </TabItem>
  <TabItem label="CLI">
    ```bash
    npm install -g @lgtm/cli
    lgtm auth login
    ```
  </TabItem>
  <TabItem label="SDK">
    ```bash
    npm install @lgtm/shared
    ```
  </TabItem>
</Tabs>

## Core capabilities

| Capability | Description |
|---|---|
| **Test case management** | Suites, sections, steps, shared steps, priorities, and tags |
| **Test execution** | Runs with per-case and per-step results, status tracking, and log capture |
| **Defect tracking** | Severity, priority, resolution workflow, and traceability to test cases |
| **Team RBAC** | Workspaces with owner/admin/member/viewer roles and per-team permissions |
| **Fine-grained tokens** | API tokens with resource-level scoping and optional project restrictions |
| **CI/CD integration** | Playwright reporter + CLI for fully automated test workflows |
| **File attachments** | Screenshots, videos, and documents on test cases, results, and defects |
| **Threaded comments** | Discussions with @mentions, reactions, and resolve/unresolve |
